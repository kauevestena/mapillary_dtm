Core Directives

Strict Enforcement: This project strictly adheres to production-first methodologies.
  • No Mocks Allowed: All tests and pipeline runs must utilize actual models, real data, and functioning reconstruction engines. Synthetic fallbacks are prohibited.
  • No Parallel Implementations: Testing and QA scripts must inherently execute the core pipeline functions rather than implementing disconnected parallel logic.
